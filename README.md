# Sudoku Generator and Solver 🧩

A simple and interactive Sudoku puzzle generator and solver built using HTML, CSS, and JavaScript.

## 🚀 Features

- ✅ Generate new Sudoku puzzles
- ✍️ Fill in the puzzle using keyboard input
- ✅ Submit your answer
- 🧠 Check if your solution is correct
- 🪄 Reveal the correct solution
- 🔒 Prevents solving/checking before submission

## 🛠 Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **No frameworks or libraries used** – pure vanilla JS!

## 🧩 How to Use

1. Click on **"Generate Puzzle"** to get a new Sudoku.
2. Enter numbers (1–9) in the empty cells.
3. Click **"Submit Answer"** when you're done.
4. Use **"Check Solution"** to validate your answer.
5. Or click **"Show Solution"** to reveal the correct solution.
